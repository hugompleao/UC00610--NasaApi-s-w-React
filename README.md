# Projeto NASA

Este projeto foi desenvolvido como avaliação final de um módulo do curso de Programação e Sistemas de Informação na ATEC. É uma aplicação web interativa construída com React e Vite que consome várias APIs da NASA para visualizar dados e imagens astronómicas.

## Funcionalidades

*   **APOD (Astronomy Picture of the Day):** Veja a imagem ou vídeo astronómico do dia escolhido pela NASA, com explicações detalhadas.
*   **Mars Photos:** Explore fotos reais da superfície de Marte capturadas pelos rovers (Curiosity, Opportunity, Spirit) da NASA.
*   **Asteroids (NeoWs):** Monitore asteroides próximos da Terra, com dados sobre tamanho, velocidade e perigo potencial.
*   **EONET (Earth Observatory Natural Event Tracker):** Acompanhe eventos naturais em tempo real, como incêndios, tempestades e vulcões.
*   **EPIC (Earth Polychromatic Imaging Camera):** Visualize imagens diárias da Terra capturadas pelo satélite DSCOVR.
*   **NASA Image Library:** Pesquise e explore a vasta biblioteca de imagens e vídeos da NASA sobre diversos temas espaciais.

## Tecnologias Utilizadas

*   **[React](https://react.dev/):** Utilizado para criar uma interface de utilizador dinâmica e modular.
*   **[Vite](https://vitejs.dev/):** Responsável por iniciar o servidor de desenvolvimento e otimizar a aplicação para produção.
*   **[Bootstrap](https://getbootstrap.com/) & [React-Bootstrap](https://react-bootstrap.netlify.app/):** Garantem um design moderno e responsivo, adaptável a diferentes tamanhos de ecrã (telemóveis, tablets e monitores).

## Configuração da API e DEMO_KEY

Para com a comunicação com a API pública da NASA, a aplicação utiliza uma chave de demonstração é necessária uma chave de demonstração (DEMO_KEY), contudo para uma melhor interação disponibilizo uma chave que disponibiliza pesquisas sem limites de utilização.

## Como executar o projeto

1.  Instale as dependências:

    ```bash
    npm install
    ```

2.  Execute o servidor de desenvolvimento:

    ```bash
    npm run dev
    ```
